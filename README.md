## Usage

1. Open `index.html` in your web browser
2. Use either the on-screen buttons or keyboard:
   - Numbers: 0-9 keys
   - Operators: +, -, *, /
   - Enter: Calculate result
   - Escape: Clear display
   - Memory shortcuts:
     - Ctrl + M: Memory Recall
     - Ctrl + Shift + M: Memory Store
     - Ctrl + Alt + M: Memory Clear
